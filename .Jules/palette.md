## 2026-03-05 - [Accessible Hamburger Menu]
**Learning:** Interactive elements like hamburger menus should be implemented using semantic `<button>` tags instead of `<div>`s to ensure they are discoverable and usable by screen readers and keyboard users. Adding an `aria-label` provides necessary context, and a clear `:focus-visible` state ensures navigation visibility.
**Action:** Always prefer semantic HTML elements for interactive components and include ARIA labels and focus indicators as a standard practice.
