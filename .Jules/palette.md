## 2025-03-12 - [Carousel Keyboard Navigation & Accessibility]
**Learning:** Interactive components like carousels often lack keyboard support in custom implementations. Adding Arrow key navigation and proper ARIA roles/labels makes them usable for a wider range of users.
**Action:** Always check for keyboard listeners and ARIA attributes in custom JS-driven UI components.

## 2025-03-12 - [Semantic Buttons & CSS Resets]
**Learning:** Converting interactive divs to <button> tags improves accessibility but can break layout due to default browser styles.
**Action:** Apply a consistent CSS reset (background, border, padding, font-family) when moving to semantic buttons.
